# Optimering

Bruk derivasjon til å finne ekstremalpunkter:
1. Finn kritiske punkter hvor $f'(x) = 0$
2. Bruk andrederiverte-testen: 
   - $f''(x) > 0$ → minimum
   - $f''(x) < 0$ → maksimum
